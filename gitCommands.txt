Git Status : To check what file are in which status like whether they are added to local environment or not.

Git init : To initialise an empty repository.

Git checkout -b "abc" : To create a new branch with name abc.

Git checkout abc : To switch from other branch to abc branch.

Git add "file1" : To add the selected file file1 to the local environment.

Git add . : To select all the files present in the folder to add in local environment.

Git Commit -m "My Message" : To Commit files in the selected branch with the message My Message.

Git push -u origin main : To push directly in the main branch.

Git push -u origin abc : To push int the abc branch.
